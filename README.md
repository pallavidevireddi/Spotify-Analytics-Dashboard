# 🎵 Spotify Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Spotify Analytics Dashboard built using Microsoft Power BI. The dashboard analyzes thousands of Spotify tracks to uncover insights into artists, albums, genres, popularity, and audio features.

The objective of this project is to demonstrate data cleaning, data modeling, DAX, and dashboard design skills while presenting meaningful business insights through interactive visualizations.

## 🎯 Business Questions Answered

- How many tracks, artists, and albums are in the dataset?
- Which artists have the highest number of tracks?
- Which albums have the highest popularity?
- How has track popularity changed over time?
- Which genres have the highest average popularity?
- What is the average danceability, energy, and duration of tracks?

  - Album

  - Total Tracks

  - Average Popularity
  - Average Danceability

  - Average Energy
